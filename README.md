# Stock Price Tracker & Analyzer
## Overview
Stock Price Tracker & Analyzer is a web application that fetches and displays **real-time stock prices** using financial APIs. It features a **clean, user-friendly interface** for monitoring market trends and analyzing stock performance.

## Features
- Real-time stock price updates
- Interactive and responsive UI
- Search and track multiple stocks
- Lightweight and fast

## Technologies Used
- HTML, CSS, JavaScript
- Python (Flask) for backend (`app.py`)
- Financial APIs for live stock data
